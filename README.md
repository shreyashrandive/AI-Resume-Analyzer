# AI Resume Analyzer

An AI-powered Resume Analyzer built using Python and Streamlit.

## Features

- Resume PDF Upload
- Resume Text Extraction
- Skill Detection
- ATS Score Calculation
- Missing Skills Analysis
- Resume Rating
- Suggestions for Improvement
- Interactive Pie Chart Analytics

## Technologies Used

- Python
- Streamlit
- PyPDF2
- Pandas
- Plotly
